About

With this site you can easily check if a site is down or if you have one of the following most common problems:

  • Browser problem: Try clearing your browser cache. Hit ctrl+shift+r.
    Also, you can try to disable your ad-blocker and refresh the page.
  • Router's problem: Restart your router.
  • DNS Problem: It may happen that your host blocks that domain.
    Try to use an alternative dns.

How does it work?

Isitdown.site will try to connect to port 80 on the address requested and report the request result. The site will appear as online, even if it returns a bad response.

Command line lover?

Check if a site is down via command line, as simple as this:

curl https://isitdown.site/api/google.com
